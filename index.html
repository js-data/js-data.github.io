JSData

JSData is a framework-agnostic, datastore-agnostic ORM/ODM for Node.js and the Browser.

Adapters allow JSData to connect to various data sources such as Firebase, MySql, RethinkDB, MongoDB, localStorage, Redis, a REST API, etc. With JSData you can re-use your Models between environments, keep your data layer intact when transitioning between app frameworks, and work with a unified data API on the server and the client. JSData employs conventions for rapid development, but allows for endless customization in order to meet your particular needs. Think of it as the "Twitter Bootstrap" of JavaScript data layers.

To get started, check out js-data.io!
